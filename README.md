## Runnable scripts/commands: what are the main commands that you run to get the results
For the dataset:

    wget https://zenodo.org/record/1175282/files/TNBC_NucleiSegmentation.zip -O dataset.zip
    unzip dataset.zip
    
To Run Model:
    
    python3 train.py
